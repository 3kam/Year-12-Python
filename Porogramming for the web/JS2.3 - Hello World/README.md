# JavaScript Lesson 2.3 — Hello World

This project contains all parts of **Lesson 2.3: Hello World**, demonstrating basic JavaScript concepts such as alerts, string concatenation, new lines, variables, and user input.

### **Part 4**

---

## Description
Each part builds upon the previous one:
- **Part 1:** Displays a simple “Hello, World” message.  
- **Part 2:** Introduces multi-line text (`\n`) and string concatenation (`+`).  
- **Part 3:** Uses variables and `prompt()` for user input.  
- **Part 4:** Adds conditional logic (`if` / `else if`) for multilingual greetings.
